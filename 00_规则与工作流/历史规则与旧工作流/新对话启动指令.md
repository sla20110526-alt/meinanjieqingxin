# 新对话启动指令

用途：每次在这个工作区新开一个短剧项目对话时，把下面的启动口令复制到第一条消息里，让我准确调用 V7.1 统一规则。

## 最推荐启动口令

```text
请先读取当前工作区里的《短剧AI视频生产规则_V7.1_统一版.md》，并把它作为本对话唯一活动执行规则。

读取后请简要确认你已理解：
1. 固定工作流
2. 创作身份只进前置分析和检查层
3. 资产类型和资产编号规则
4. Seedance2.0 单文本框结构
5. 纯 Prompt 所见即所得最高优先级
6. 全局生成约束禁止字幕和音乐
7. 光影全局纪律与局部可见结果
8. 覆盖镜头池不写说明性文字
9. 正式 Seedance2.0 Prompt 生成前必须先做本组镜头策略
10. 正式 Seedance2.0 Prompt 生成后必须执行 V7.1 证据式二次规则检查
11. Seedance 实测失败后必须记录失败类型并判断是否沉淀为规则补丁

后续我会用你做短剧剧本分析、资产拆解、资产生图 Prompt、分镜表、Seedance2.0 分镜 Prompt 和覆盖镜头池。请默认按 V7.1 推进，并在每个关键阶段完成后等我确认。
```

## 新对话原则

新对话不会自动继承上一个对话的上下文。每个新对话第一句话必须明确要求读取 V7.1。

如果我没有先读取 V7.1 就开始分析，你可以打断：

```text
先暂停。请先读取《短剧AI视频生产规则_V7.1_统一版.md》，再继续。
```
